## About the project

Laravel 9, Inertia, Vite 3, Vuejs 3, Vuetify 3 Admin Template

## Required technologies

- PHP 8.0
- Laravel 9
- Inertia
- Vite 3
- Tailwind CSS 3
- Alpine JS 3
- Vue 3
- Vuetify 3
- Mariadb 10.4
- NodeJS 16
- Apache

## Create database

- DB_CONNECTION=mysql
- DB_HOST=ddev-vuetify-db
- DB_PORT=3306
- DB_DATABASE=laravel
- DB_USERNAME=root
- DB_PASSWORD=root

## Install Project

- git clone https://github.com/devrazec/vuetify.git
- composer install
- php artisan migrate:fresh
- php artisan db:seed
- npm install
- npm run build
